# Library-Management-System
Framework : ASP.NET Core MVC <br> 
Front-end : Bootstrap

Description: A well organized software solution for library which helps to provide information on any book present in the library to the admin of the library.

Some Features are :
- Create new book <br>
- Edit book <br>
- Delete book <br>
- Display book details <br>
- Create new author <br>
- Edit author <br>
- Delete author <br>
- Display author details <br>
- Create new customer <br>
- Edit customer information <br>
- Delete customer <br>
- Display customer details <br>
- Lend book and return book <br>
- Books count <br>
- Authors count <br>
- Customer count <br>
- Admin panel <br>
